# Convert-Numbers-to-Japanese
Converts Arabic numerals, or 'western' style numbers to a Japanese context.

Usage:
``` bash
Convert(number,type)
``` 
types; "kanji", "hiragana", "romaji"

Examples:
``` bash
Convert(20.8,"romaji")
Convert(2000,"hiragana")
Convert(458,"kanji")
``` 

Can also convert Kanji to 'western' number using:
``` bash
ConvertKanji(kanji)
``` 

Examples:
``` bash
ConvertKanji("二十点五")
ConvertKanji("二十五")
```
